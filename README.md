🇮🇳 Bharat Clock

A modern and responsive Bharat Clock built using React, HTML, CSS, and JavaScript. This application displays the current Indian Standard Time (IST) with a clean UI and real-time updates.

🚀 Features
🕒 Real-time clock (updates every second)
🇮🇳 Displays Indian Standard Time (IST)
⚛️ Built using React components
🎨 Clean and responsive UI
⚡ Lightweight and fast

🛠️ Tech Stack
React.js – UI & Component Logic
HTML5 – Structure
CSS3 – Styling
JavaScript (ES6+) – Functionality

🎯 How It Works
Uses JavaScript Date API to fetch current time
Updates every second using setInterval
React re-renders the UI dynamically
